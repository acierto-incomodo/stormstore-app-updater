# Changelog - StormStore v1.2.11

## 🎮 Discord Rich Presence (RPC)

- **Nueva Integración:** Se ha añadido soporte para Discord RPC.
  - Muestra el estado "Explorando aplicaciones" o el juego específico que estás viendo.
  - Soporte para el modo **StormVortex**.
  - Iconos dinámicos según la plataforma (Steam, Epic, StormStore).

## 📱 Aplicaciones

- **Ordenación:** La lista de aplicaciones ahora está ordenada alfabéticamente (manteniendo StormStore al inicio) para facilitar la navegación.

## 🛠️ Cambios Técnicos

- **Dependencias:** Añadido `discord-rpc`.
- **Versión:** Actualizada a v1.2.11.
